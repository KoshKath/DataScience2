Создание рекуррентного автоэнкодера, предназначенного для обработки и восстановления данных временных рядов, таких как финансовые показатели или сенсорные данные.
  Для выоплнения ДЗ был использован датасет Human Activity Recognition Using Smartphones Data Set с целью распознавания
  активности человека по данным с сенсоров смартфона.
  Обучение GRU-автоэнкодера проходило на 70 эпохах, которые показывают, что:
  начальный loss был 1.0636, а к концу обучения составил 0.354;
  val_loss с 0.8021 снизился до 0.3187.
  При этом можно проследить небольшие колебания loss в периоод обучения (незначительные колебания).
  В ДЗ также решалась задача классификации классов с помощью классификатора XGBClassifier. 
  Достигнута точность 0.91. По реультатам работы модель немного путает классы ходьбы между собой, классы сидения и стояния.
